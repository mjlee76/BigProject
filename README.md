# 서비스 소개

## 가상환경 설정

### conda
```
$ conda create -n <환경명> python=3.11
```
- ctrl + shift + p : 인터프리터로 <환경명> 설정

```
$ pip install -r requriment.txt
```

### vs code
```
$ python -m venv venv
```
- ctrl + shift + p : 인터프리터 설정

```
$ cd venv Scripts

$ activate venv

$ pip install -r requriment.txt
```
